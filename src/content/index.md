Svelte Simple \* is a suite of libraries designed to simplify the development process of Svelte applications. Its primary focus is on ease of use, making it an ideal choice for developers of all skill levels.

By providing a collection of simple, yet powerful tools, Svelte Simple enables developers to quickly and easily build high-quality web applications. Whether you're building a simple website or a complex web application, Svelte Simple has the tools you need to get the job done.

## Tools

- **Svelte Simple Form**. A simple yet powerful, lightweight form handling library for Svelte 5 or above
- **Svelte Simple Query**. A simple yet powerful, lightweight data query library for Svelte 5 or above
- **Svelte Simple Lang**. A lightweight library to manage multi-language (i18n) for Svelte 5 or above. It comes with a simple setup and much features.
